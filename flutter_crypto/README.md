# fluttercrypto

A dart crypto library. 
当前仅支持DES 算法 ECB CBC模式
相比插件调用 无需异步 毫秒级响应
对比flutter_des  10次耗时 1400+ms    本库仅需要 不到100ms  调用次数越少差异越明显
